# a5druck

## Description
Simple CLI-Tool to create printable a4-paper containing two duplicated a5-paper.

## Usage
```console
python3 a5druck.py a5file.pdf
```

or 

```console
a5druck a5file.pdf
```

if it exists in bin, e.g.

```console
sudo cp a5druck.py /usr/bin/a5druck
```

## Dependencies 

Maybe you will need pypdf

```console
pip3 install pypdf
```
